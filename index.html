Ayuda Git



Comandos

Se debe inicializar el repositorio desde consola



git init

Inicia el repositorio o carpeta donde se encuentran los archivos que vamos a modificar.




git add

Agrega el archivo al cual le vamos a dar seguimiento en las versiones.

Ejemplo:
  add archivo.txt

//Tambien se pude usar el comando git add . para agregar todos los archivos.




git commit -m "mensaje"

Agrega el archivo al cual le vamos a dar seguimiento en las versiones.

Ejemplo:
  add archivo.txt




git status

Si se ha hecho un cambio y no se ha añadido, aqui va a salir, si no hemos hecho modificaciones aqui nos mostrará de igual forma.




git show

Muestra todos los cambios historicos hechos, como las lineas que han cambiado, cuando y quien lo cambiaron.




git log

Se usa como git log «archivo» para ver la historia completa de un archivo.




git push

Para subir nuestros archivos al servidor remoto.
